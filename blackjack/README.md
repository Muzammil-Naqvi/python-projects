# 🃏 Blackjack Game (Python)

## 📌 Description

A command-line based Blackjack game built using Python, implementing core game mechanics and user interaction.

## 🚀 Features

* Player vs Dealer gameplay
* Hit / Stand options
* Random card generation
* Score calculation based on Blackjack rules

## 🛠️ Technologies Used

* Python

## ▶️ How to Run

```bash
python main.py
```
